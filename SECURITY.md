# Security Policy

DNS Security TXT is a [disclose.io](https://disclose.io) project. If you find a security issue in this repository or on dnssecuritytxt.org, please report it via:

- **Email:** security@disclose.io
- **Web:** https://disclose.io/security/

This project practices what it preaches — see the live DNS Security TXT record:

```
dig +short TXT _security.dnssecuritytxt.org
```

and the site's [security.txt](https://dnssecuritytxt.org/.well-known/security.txt) (RFC 9116).
