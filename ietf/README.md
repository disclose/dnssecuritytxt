# DNS Security TXT Internet-Draft

This directory contains the proposed Standards Track Internet-Draft for DNS-based discovery of vulnerability-reporting contacts and policies.

## Files

- `draft-ellis-dns-security-contact-00.xml` — authoritative RFCXML source
- `draft-ellis-dns-security-contact-00.txt` — rendered plain-text Internet-Draft
- `draft-ellis-dns-security-contact-00.html` — rendered HTML Internet-Draft
- `SUBMISSION-NOTES.md` — design decisions, open DNSOP questions, submission checklist, and suggested mailing-list introduction

## Rendering

Install the current IETF `xml2rfc` tool and run:

```sh
xml2rfc --v3 --text --html draft-ellis-dns-security-contact-00.xml
```

The checked-in text and HTML outputs must be regenerated whenever the RFCXML source changes.

## Status

This is an individual Internet-Draft under development. It has not yet been submitted, adopted by an IETF working group, or approved for publication as an RFC.
